发表于 2018-02-26

概要

条件控制：if、when

if

  • 作为语句，包含一个布尔表达式和一条或多条语句

  • 作为表达式，即它会返回一个值

when

  • 类switch操作符

    • 常规用法

    • 很多分支需要用相同的方式处理，则可以把多个分支条件放在一起，用逗号分隔

    • 使用 in（在）或!in（不在） 运算符来检测某个数字是否在指定区间或集合内，区间格式为 x..y

    • 检测一个值是（is）或者不是（!is）一个特定类型的值

  • 取代 if-else if链。如果不提供参数，所有的分支条件都是简单的布尔表达式，而当一个分支的条件为真时则执行该分支
