kotlin-extend

发表于 2018-03-04

概要

  • Kotlin 中所有类都继承该 Any 类，它是所有类的超类
  • 如果一个类要被继承，可以使用 open 关键字进行修饰

构造函数

  • 子类有主构造函数
    基类必须在主构造函数中立即初始化

  • 子类没有主构造函数
    在子类的每一个二级构造函数中用 super 关键字初始化基类

重写

在基类中，使用fun声明函数时，此函数默认为final修饰，不能被子类重写。如果允许子类重写该函数，那么就要手动添加 open 修饰它, 子类重写方法使用 override 关键词

如果继承的两个基类中存在相同的方法，则必须要重写该方法，重写时可采用 super 范型去选择性地调用父类的实现。

属性重写

  • 使用 override 关键字

  • var可以属性重写val属性，但是反过来不行

  • 可在主构造函数中使用 override 关键字作为属性声明的一部分

kotlin-returns

发表于 2018-02-27

概要

Kotlin 有三种结构化跳转表达式：

  • return：默认从最直接包围它的函数或者匿名函数返回。
  • break：终止最直接包围它的循环。
  • continue：继续下一次最直接包围它的循环。

标签

在 Kotlin 中任何表达式都可以用标签（label）来标记。
标签的格式为标识符后跟 @ 符号，例如：abc@、fooBar@都是有效的标签。

Break 和 Continue 标签

标签限制的 break 跳转到刚好位于该标签指定的循环后面的执行点；
标签限制的 continue 继续标签指定的循环的下一次迭代。

Return标签

标签限制的 return 允许我们从外层函数返回，最重要的一个用途就是从 lambda 表达式中返回

  • 不使用标签的lambda 表达式

  • 使用标签的lambda 表达式

  • 使用隐式标签的lambda 表达式

  • 使用匿名函数替代的lambda 表达式

  • 使用标签直接返回值

kotlin-basic-types

发表于 2018-02-22

概要

基本类型：数字、字符、布尔值、数组与字符串。

在 Kotlin 中，所有东西都是对象，即可以在任何变量上调用成员函数和属性。

数字

没有隐式转换，不支持八进制，可使用下划线使数字常量更易读

表示方式

  • 十进制: 123 / 1_2_3
  • Long 类型用大写 L 标记: 123L /1_2_3L
  • 十六进制: 0x0F / 0x_0F
  • 二进制: 0b00001011 / 0b_0_0_001_011
  • double：123.5、123.5e10 / 1_23.5
  • Float 用 f 或者 F 标记: 123.5f /1_23.5f

显示转换

  • toByte(): Byte
  • toShort(): Short
  • toInt(): Int
  • toLong(): Long
  • toFloat(): Float
  • toDouble(): Double
  • toChar(): Char

运算

  • shl(bits) – 有符号左移 (Java 的 <<)
  • shr(bits) – 有符号右移 (Java 的 >>)
  • ushr(bits) – 无符号右移 (Java 的 >>>)
  • and(bits) – 位与
  • or(bits) – 位或
  • xor(bits) – 位异或
  • inv() – 位非

浮点数比较

  • 相等性检测：a == b 与 a != b
  • 比较操作符：a < b、 a > b、 a <= b、 a >= b
  • 区间实例以及区间检测：a..b、 x in a..b、 x !in a..b

字符

  • 用 Char 类型表示，用单引号括起来
  • 特殊字符可以用反斜杠转义
  • 编码其他字符要用 Unicode 转义序列语法，如’\uFF00’

布尔

  • 用 Boolean 类型表示，有两个值：true 和 false
  • 内置的布尔运算
    • || – 短路逻辑或
    • && – 短路逻辑与
    • ! - 逻辑非

数组

  • 用 Array 类来表示
  • 使用库函数 arrayOf() 来创建一个数组并传递元素值给它，这样 arrayOf(1, 2, 3) 创建了 array [1, 2, 3]
  • 库函数 arrayOfNulls() 可以用于创建一个指定大小的、所有元素都为空的数组

字符串

  • 用 String 类型表示
  • 字符串是不可变的
  • 字符串的元素可以使用索引运算符访问
  • 可以用 for 循环迭代字符串

字符串字面值

分为：转义字符串、原生字符串和文本字符串

  • 转义字符串

  • 原生字符串：使用三个引号（”””）分界符括起来，内部没有转义并且可以包含换行和任何其他字符

字符串模板

字符串可以包含模板表达式 ，模板表达式以美元符（$）开头
